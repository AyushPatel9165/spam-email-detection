 # Spam Email Detection Project

## Overview
This project detects whether an email is spam or not using Natural Language Processing (NLP) and Machine Learning classification models such as Naive Bayes or Logistic Regression.

## Features
- Classify emails as spam or not spam
- Data preprocessing and cleaning
- NLP techniques for feature extraction
- Model training and evaluation
- Beginner-friendly ML project

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK (Natural Language Toolkit)
- Pickle
